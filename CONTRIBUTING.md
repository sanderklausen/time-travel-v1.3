# Contributing to Time Travel v1.3

Thank you for your interest in contributing! This project is a dream brought to life, and we appreciate your enthusiasm.

## How to Contribute

1. **Fork** the repository
2. **Create a branch** (`git checkout -b feature/amazing-feature`)
3. **Make your changes** and commit (`git commit -m 'Add amazing feature'`)
4. **Push to the branch** (`git push origin feature/amazing-feature`)
5. **Open a Pull Request**

## Code Style

- Follow PEP 8 guidelines
- Use meaningful variable names
- Add comments for complex logic
- Test your changes before submitting

## Reporting Issues

Found a bug? Please [open an issue](https://github.com/sanderklausen/time-travel-v1.3/issues) with:
- A clear description
- Steps to reproduce
- Expected vs actual behavior

## Questions?

Feel free to open a discussion or issue with your questions!
