# Time Travel v1.3

> A surreal dream brought to life. A fully functional time travel interface built in Python.

A recreation of the Time Travel software interface from a dream, featuring a beautiful vintage aesthetic with date/time controls, location input, and the mysterious "Sqwimble" parameter.

## Features

✨ **Dream-Inspired Interface**
- Elegant vintage UI with warm color palette
- Date/Time selection (Day, Month, Year, CE/BCE)
- Location input (Latitude, Longitude)
- Sqwimble slider (-100 to 100 scale)
- Deterministic and Assume Form checkboxes
- Interactive Start button

🎨 **Design**
- Professional tkinter-based GUI
- Responsive layout
- Custom color scheme (#ECE9D8, #4169e1)
- High-quality image support

## Requirements

- Python 3.7+
- tkinter (included with Python)
- Pillow 9.0.0+

## Installation

### Clone the Repository

```bash
git clone https://github.com/sanderklausen/time-travel-v1.3.git
cd time-travel-v1.3
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## Usage

Run the application:

```bash
python timetravel.py
```

### The Sqwimble Parameter

Adjust the Sqwimble slider from -100 to 100 to control temporal quantum fluctuations.

### Start

Try clicking the **Start** button to experience a burst of temporal energy!
